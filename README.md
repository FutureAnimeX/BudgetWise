# BudgetWise App

A simple and intuitive budget tracking application designed to help users manage their finances, track expenses, and reach their savings goals with ease.

---

##  Features

* **Expense Tracking:** Categorize and log daily spending in real-time.
* **Budget Planning:** Set monthly limits for different categories like groceries, entertainment, and rent.
* **Visual Analytics:** View spending habits through clean, interactive charts and graphs.
* **Savings Goals:** Define and track progress toward specific financial milestones.
* **Secure Data:** Keep your financial information private with local storage or encrypted cloud sync.
